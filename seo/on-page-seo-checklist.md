# On-Page SEO Checklist — Mesquite Manufacturing (WordPress)

**Site:** mesquitemanufacturing.com (WordPress)  
**Recommended plugins:** Yoast SEO or Rank Math (pick one, not both)

---

## 🔴 Critical — Fix First

### Title Tags
- [ ] Homepage: `Mesquite Manufacturing | CNC Machining & Milling | Mesquite, TX` (under 60 chars)
- [ ] Services page: `CNC Machining Services | Precision Milling Texas | Mesquite Manufacturing`
- [ ] Each service sub-page: `[Service Name] in Texas | Mesquite Manufacturing`
- [ ] Blog posts: `[Post Title] | Mesquite Manufacturing` (target keyword near the front)
- [ ] Contact/RFQ page: `Get a Free CNC Machining Quote | Mesquite Manufacturing`
- [ ] Never duplicate title tags across pages

### Meta Descriptions
- [ ] Write unique meta descriptions for every page (150–160 chars)
- [ ] Include primary keyword naturally
- [ ] Include a call to action ("Get a free quote", "Learn more")
- [ ] Homepage example: `Precision CNC machining & milling in Mesquite, TX. 4th axis CNC, ±0.001″ tolerances, 3D printing & custom assembly. 24/7 availability. Free RFQ today.`
- [ ] Never leave meta descriptions blank (Google will auto-generate — often badly)

### H1 Tags
- [ ] Every page has exactly ONE H1 tag
- [ ] H1 contains the primary keyword for that page
- [ ] Homepage H1: "CNC Machining & Milling Built Around You" (or similar with Texas/Mesquite in text)
- [ ] Do not use H1 for decorative headings

### Header Hierarchy
- [ ] H1 → H2 → H3 structure (never skip levels)
- [ ] H2s organize major sections
- [ ] H3s organize sub-points within sections
- [ ] At least 3 H2 tags per long-form service page

---

## 🟡 Important — Do Within 30 Days

### Image Optimization
- [ ] All images have descriptive alt text (not "image001.jpg")
- [ ] Alt text describes the image AND naturally includes keywords where relevant
  - Good: `alt="4th axis CNC machining aluminum part on Haas machine"`
  - Bad: `alt="machine"` or `alt="IMG_4023"`
- [ ] Images compressed to <200KB where possible (use TinyPNG or Smush plugin)
- [ ] Use WebP format for new images (WordPress 5.8+ supports natively)
- [ ] Filename includes keywords: `cnc-machining-texas.jpg` not `DSC001.jpg`

### URL Structure
- [ ] Short, keyword-rich URLs
  - Good: `/services/cnc-machining`
  - Bad: `/services/?p=12` or `/precision-cnc-machining-and-milling-services-in-texas/`
- [ ] No dates in URLs for evergreen content (avoid `/2023/11/cnc-machining/`)
- [ ] All lowercase, hyphens not underscores
- [ ] Set canonical URL on every page to avoid duplicate content

### Internal Linking
- [ ] Every service page links to the RFQ/quote page with CTA
- [ ] Blog posts link to relevant service pages (2–3 internal links minimum)
- [ ] Homepage links to each main service page
- [ ] Use descriptive anchor text: "CNC machining services" not "click here"
- [ ] Create a `resources` or `blog` page that links to all blog posts
- [ ] Consider a siloed structure:
  - Homepage → /services → /services/cnc-machining
  - /services/cnc-machining → blog posts about CNC machining

### Page Speed & Core Web Vitals
- [ ] Install and configure caching plugin (WP Rocket, W3 Total Cache, or LiteSpeed Cache)
- [ ] Enable Gzip/Brotli compression at server level or via caching plugin
- [ ] Minify CSS, JS, and HTML
- [ ] Lazy-load images below the fold
- [ ] Use a CDN (Cloudflare free tier works well)
- [ ] Test with Google PageSpeed Insights (aim for 80+ on mobile)
- [ ] Core Web Vitals targets:
  - LCP (Largest Contentful Paint): < 2.5s
  - FID / INP: < 100ms
  - CLS (Cumulative Layout Shift): < 0.1

### Mobile Optimization
- [ ] Site is fully responsive (test on iPhone and Android)
- [ ] Touch targets (buttons) are at least 44px tall
- [ ] Text is readable without zooming (min 16px body font)
- [ ] RFQ forms work on mobile with minimal friction
- [ ] Phone number is tap-to-call (`<a href="tel:+1XXXXXXXXXX">`)

## 🟢 Schema & Technical SEO

### Schema Markup (see schema-markup-additions.json)
- [ ] LocalBusiness / ManufacturingBusiness schema on homepage
- [ ] FAQPage schema on homepage or FAQ page
- [ ] Service schema on each service page
- [ ] BreadcrumbList on all interior pages
- [ ] AggregateRating once reviews are collected
- [ ] Test schema at: https://search.google.com/test/rich-results

### XML Sitemap
- [ ] Sitemap auto-generated by Yoast/Rank Math at `/sitemap.xml`
- [ ] Submitted to Google Search Console
- [ ] Submitted to Bing Webmaster Tools
- [ ] Excludes tag pages, author archives, search result pages
- [ ] Updated automatically when new content is published

### Robots.txt
- [ ] `/robots.txt` exists and is not blocking search engines
- [ ] Points to sitemap: `Sitemap: https://mesquitemanufacturing.com/sitemap.xml`
- [ ] Block: `/wp-admin/`, `/wp-login.php`, `/xmlrpc.php`

### Canonical Tags
- [ ] `rel="canonical"` on every page pointing to itself (Yoast handles this)
- [ ] If using UTM parameters in any URLs, canonical points to clean URL
- [ ] www vs non-www: pick one and 301-redirect the other

### HTTPS & Security
- [ ] Site runs on HTTPS (SSL active) ✅
- [ ] HTTP → HTTPS 301 redirect active
- [ ] Mixed content warnings resolved (no HTTP assets on HTTPS pages)

## 🟢 Content Quality

### Per Page Content Quality
- [ ] Every service page has at least 500 words of unique, useful content
- [ ] Content answers: What? Who? Why? How? How much?
- [ ] Include specifics: materials handled, tolerances, equipment, turnaround
- [ ] Avoid: keyword stuffing, duplicate content, thin pages
- [ ] Each page has a clear, prominent CTA (RFQ button, phone, contact form)

### Blog Post Standards
- [ ] Minimum 1,200 words per post (1,800+ for competitive keywords)
- [ ] Include at least one image with descriptive alt text
- [ ] Include a table of contents for long posts (use a plugin or manual anchor links)
- [ ] End every post with an RFQ CTA
- [ ] Tag posts with relevant categories (CNC Machining, 3D Printing, Texas Manufacturing, etc.)

### Contact / RFQ Page
- [ ] Clear headline: "Get a Free CNC Machining Quote"
- [ ] Short reassurance copy (24-hour response, no obligation)
- [ ] Simple form (name, email, phone, service, description, file upload)
- [ ] Accept: .STEP, .IGES, .DXF, .DWG, .PDF
- [ ] Thank-you page after form submission (with Google Analytics conversion tracking)
- [ ] Phone number displayed prominently

## WordPress-Specific Checklist

- [ ] Yoast SEO or Rank Math installed and configured
- [ ] Set site language to English (US)
- [ ] Disable Settings → Reading → "Discourage search engines" (common staging mistake)
- [ ] Configure permalink structure: `/%postname%/` (Settings → Permalinks)
- [ ] Delete unused themes and plugins
- [ ] Keep WordPress, themes, and plugins updated (security + speed)
- [ ] Set up Google Search Console: https://search.google.com/search-console
- [ ] Set up Google Analytics 4 (GA4): https://analytics.google.com
- [ ] Connect GSC to GA4 for combined insights
