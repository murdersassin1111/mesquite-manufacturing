## Key Priorities

### 🔴 Week 1 — Quick Wins (highest ROI, lowest effort)
1. **Google Business Profile** — Claim, verify, and fully optimize. This is the single biggest driver of local search traffic.
2. **Canonical NAP** — Define your exact Name/Address/Phone format. Use it everywhere.
3. **Bing Places + Apple Maps** — Sync from GBP. Takes 30 minutes. Covers iOS users.
4. **ThomasNet free profile** — The #1 industrial B2B directory. Engineers search here.
5. **Foursquare for Business** — Submitting here pushes to 100+ downstream aggregators.
6. **Yelp + LinkedIn Company Page** — High DA, B2B visibility.

### 🟡 Weeks 2–4 — Citation Building
- Submit to all Tier 1 and Tier 2 directories in `backlink-targets.md`
- Focus: manufacturing directories (Mfg.com, Maker's Row, IndustryNet, GlobalSpec)
- Local: Mesquite Chamber, Dallas BBB, DFW manufacturing associations
- Track all submissions in a spreadsheet (site, date, status, login, live URL)

### 🟢 Month 2 — Content & Outreach
- Publish first blog post (recommend: "How to Write an RFQ" — captures buyers at decision point)
- Begin guest post outreach to The Fabricator, Modern Machine Shop
- Start review request emails to past clients
- Add schema markup to WordPress site (copy from `schema-markup-additions.json`)

### 🔵 Month 3+ — Scale
- Publish 2 blog posts/month (follow `blog-content-plan.md` calendar)
- Build location-specific service pages (/dallas, /fort-worth, etc.)
- Monitor Google Search Console for ranking progress
- Expand directory submissions to Tier 3 targets

## Canonical NAP Reminder

Every directory, listing, and page footer must use IDENTICAL formatting:

```
[Phone Number]
https://mesquitemanufacturing.com
```

Pick your format and never deviate.
